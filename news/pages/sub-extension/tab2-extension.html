Catatan Untuk Markdown Preview Github Styling:
1. Setelah install extension, download file markdown-custom-style.css (klik disini).
2. Pindahkan file tersebut ke folder yang sama dimana markdown Anda ditulis.
3. Buka VSCode ➜ Settings ➜ Pilih tab workspace ➜ Ketik "markdown styles" ➜ Add item.
4. Kemudian isi Add item dengan "markdown-custom-style.css" ➜ OK.
5. Restart VSCode, buka file markdown Anda, klik icon "Live Preview" di pojok kanan atas.
6. Selesai. Sebagai catatan, gunakan SHIFT+ALT+0 untuk mengubah split editor menjadi vertical.
Tujuan kustomisasi markdown styles di atas yaitu agar width (lebar) preview markdown sama persis dengan di GitHub.

Formatter
Catatan Umum:
Beberapa Extension dijalankan melalui perintah di CTRL+Shift+P ➜ Ketik nama Extensionnya.
Catatan Untuk Beautify & Prettier:
1. Buka Settings ➜ Ketik "Format" ➜ Default Formatter (Sesuaikan) ➜ Centang "Format on Save".
2. Buka Settings ➜ Ketik "Wrap line length" ➜ Sesuaikan dengan dengan lebar monitor anda.
3. Format secara manual dengan cara ➜ Ketik "Format Document with..." ➜ Pilih Formatter.
Catatan Untuk Strict Whitespace:
Untuk memperbaiki Whitespace ➜ CTRL+Shift+P ➜ Ketik "Trim Trailing Whitespace" ➜ Enter.
